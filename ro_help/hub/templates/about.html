Despre RO help

În contextul pandemiei cauzate de virusul COVID-19, este mai important ca niciodată să fim conștienți că orice sprijin din partea noastră poate contribui la îmbunătățirea situației generale.

Code for Romania, prin programul Code for Romania Task Force, a dezvoltat aplicația RO Help ca platformă care să faciliteze solicitarea sau colectarea de resurse, fie ele materiale sau financiare precum și înregistrarea de voluntari.

RO help este destinată ONG-urilor, care pot să solicite ajutor prin intermediul aplicației, respectând următoarele recomandări:

  • specificarea tipului de resursă solicitată
  • crearea unei solicitări per resursă și per oras.

Procesul de urmat pentru cetățeni este similar. O dată ce solicitarea a fost creată și direcționată către un ONG, cetățeanului nu îi mai ramane de făcut decat să aștepte soluționarea acesteia. Când solicitarea a fost îndeplinită, aceasta va fi închisă.

RO Help este un proiect realizat de Code for Romania în parteneriat cu Guvernul României prin Autoritatea pentru Digitalizarea României și Departamentul pentru Situații de Urgență.
